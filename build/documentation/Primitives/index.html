Terraformer includes a lightweight wrapper to GeoJSON allowing for a number of methods to be used on them.

Link Primitive Back to Top

Terraformer.Primitive is the base class for all primitives. As such, it can perform most actions that every other primitive can. All other primitives inherit from Terraformer.Primitive, and thus all methods on Terraformer.Primitive are available on those primitives.

When a GeoJSON object is passed in at instantiation, it will create an object of that type of GeoJSON.

Link Instantiation Back to Top 

// create an empty primitive
var primitive = new Terraformer.Primitive();

// create a typed primitive from GeoJSON
// this will create an object of type Point
var point = new Terraformer.Primitive({ "type": "Point", "coordinates": [ 100, 1 ] });

Link Coordinate Systems Back to Top

Terraformer has the ability to switch between Web Mercator and Geographic (WGS84) coordinate systems. This is an in-place modification to the object.

// to Web Mercator
primitive.toMercator();

// to WGS84
primitive.toGeographic();

Link Bounding Box Back to Top

All primitives have the possibility of a bounding box. A bounding box returns an array 4 points: x1, x2, y1, and y2 denoting a square that encapsulates all points.

// create a Polygon
var poly = new Terraformer.Primitive({
  "type": "Polygon",
  "coordinates": [ [ [10, 10], [30, 20], [40, 30], [40, 40], [10, 10] ] ]
});

// should return [ 10, 10, 40, 40 ]
var box = poly.bbox;

All primitives have a set of geographic tools available.

Link Envelope Back to Top

This will return an envelope suitible for use with Terraformer.RTree of x, y, width, height. js // returns the convex hull of the polygon var ev = polygon.envelope();

Link Within Back to Top

This will return true if the primitive is contained or equal to the primitive passed. “`js var polygon1 = new Terraformer.Polygon([ [ [ 2, 2 ], [ 2, 3 ], [ 3, 3 ], [ 3, 2 ], [ 2, 2 ] ] ]); var polygon2 = new Terraformer.Polygon([ [ [ 1, 1 ], [ 1, 5 ], [ 5, 5 ], [ 5, 1 ], [ 1, 1 ] ] ]);

// true
polygon1.within(polygon2);

// false
polygon2.within(polygon1);

## Point

A `Point` is a single point in space.

### Instantiation
```js
    // create an empty Point
    var point = new Terraformer.Point();

    // create a Point from an Array of x (longitude) and y (latitude)
    var point = new Terraformer.Point( [ 10, 10 ] );

    // create a Point from GeoJSON
    var point = new Terraformer.Point({
      "type": "Point",
      "coordinates": [ 10, 10 ]
    });

Link Feature Back to Top

Features represent custom data (properties in the GeoJSON spec) and geographic data togather in a single object.

Link Instantiation Back to Top

You can instantiate a Feature by passing in a GeoJSON Feature or any GeoJSON geometry. But you must assign an id property for it to be valid GeoJSON. js // creates a feature from a valid GeoJSON Object. var feature = new Terraformer.Feature({"type": "Point", "coordinates": [ 10, 10 ]);

Link Contains Back to Top

If your features geometry type is Polygon or MultiPolygon you can check to see if your Feature contains a Point Geometry. Currently, it will only operate on aPoint, but does check to make sure that thePointis contained only in the Polygon area, and not within a hole. ``js // create a point var point = new Terraformer.Point([ 10, 10 ]);

// create a polygon
var feature = new Terraformer.Feature({type: "Polygon", coordinates: [ [ [ 0, 0 ], [ 0, 20 ], [ 20, 20 ], [  20, 0 ] ] ]});

// check the results
feature.contains(point); // should be true

### Intersects

```js
    // check for intersection

Link FeatureCollection Back to Top

A FeatureCollection is a wrapper around a set of features.

Link Instantiation Back to Top

You can create a FeatureCollection by passing in a GeoJSON FeatureCollection or an array of Features. js var fc = new Terraformer.FeatureCollection([feature1, feature2]);

Link Getting by ID Back to Top

Because Features must have an id to be valid GeoJSON, you can search for a specific id with the get method. js var fooFeature = fc.get("foo"); // will get the feature with the id of foo

Link GeometryCollection Back to Top

A GeometryCollection represents a set of geometries.

Link Instantiation Back to Top

You can create a GeometryCollection by passing in array of GeoJSON geometries or a GeoJSON GeometryCollection. js var gc = new Terraformer.GeometryCollection([geom1, geom2]);

Link Getting A Geometry Back to Top

Like other multi geometry objects you can use the get method to get a the geometry at a specific index as an instance of Terraformer.Primitive. js var myGeometry = gc.get(2);

Link Circle Back to Top

GeoJSON does not provide support for circles in the specification. However Terraformer provides circles as a type of Feature which has getters and setters for center, steps, and radius. The Feature contains a polygon which repesents the circe.

Link Instantiation Back to Top

You can create a Circle my passing in a GeoJSON position representing the center of a circle and the radius of the circle. By default radius will be set to 250 meters. js // create a 1000 meter circle centered on the Esri R&D Center in Portland var circle = new Terraformer.Circle([-122.6764, 45.5165], 1000); By default circle will be made of a 64 points. You can change this by passing in a third parameter. js // this "circle" will only have 16 points var circle = new Terraformer.Circle([-122.6764, 45.5165], 100, 16);

Link Moving The Circle Back to Top

If you want to move the circle to a new location you can update the center property. Changing this property will recreate the circle polygon at the new location. js // move the circle to Null Island circle.center = [0,0];

Link Changing The Radius Back to Top

Chaging the radius is similar to chaning the center of the circle. Simply set the radius property on the circle. Changing this property will recreate the circle polygon. js // make the circle really big circle.radius = 2500

Link Chagning Interpolation Back to Top

As you change and alter your circle you may need to change the detail level. Changing the steps property will alter the level of detail in your circle. js // make the circle more detailed circle.steps = 128;

Link Contains Back to Top

Since circles are polygons you can see if they contain points. js // does this circle contain a point circle.contains(point); // true or false

Link Intersects Back to Top

It is possible to check for the intersection of LineString, MultiLineString, Polygon, Circle, and MultiPolygon and Feature. Any other type of intersection will result in false being returned. js // check for intersection circle.intersects(new Terraformer.LineString([ [ 20, 10 ], [ 10, 20 ] ]));
